Hexo+github actions完成个人博客的自动部署

This is the first blog! it is created by github-actions!
本人一直对个人博客比较感兴趣，之前也用过hexo进行过博客的编写，但由于其编写需要部署hexo环境，有点麻烦，而我又想在多端编写，所以后来也就放弃了。但最近发现github actions有这个自动部署的功能，在借鉴了网上的资料之后也终于搭建一个可以自动部署的hexo。

站点演示传送门

Hexo

More info: Generating

github actions

github actions分为两个部分的配置文件，一个部分是新建博客，另一个是部署博客

新建博客

新建博客需要我们在github的仓库里 点击actions ，手动运行New Blog的action进行运行
